Unnarize

A high-performance bytecode interpreter for a dynamic, C-style scripting language.

Get Started GitHub

Features

Clean Syntax
Familiar C-style syntax with modern touches for intuitive programming.
High Performance
Bytecode VM with computed goto dispatch for maximum speed.
Module System
Hierarchical imports with relative path resolution.
Async/Await
First-class asynchronous programming support.
Structs
Custom data types with field access.
Zero Dependencies
Pure C implementation with no external libraries.

Quick Example

// Hello World
var name = "Unnarize";
print("Hello, " + name + "!");

// Function
function greet(who) {
    return "Welcome, " + who;
}

print(greet("Developer"));

Installation

git clone https://github.com/gtkrshnaaa/unnarize.git
cd unnarize
make
./bin/unnarize examples/basics/01_variables.unna

Core Libraries

Library Description
ucoreString Text manipulation, split, join, replace, regex
ucoreScraper HTML parsing, CSS selectors, web scraping
ucoreJson JSON parsing and serialization
ucoreHttp HTTP client and server, REST APIs
ucoreTimer High-precision timing and sleep
ucoreSystem File I/O, shell exec, environment
ucoreUon UON format database with streaming cursor
