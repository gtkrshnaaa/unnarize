ucoreHttp

Built-in HTTP server functionality.

Functions

FunctionDescription
listen(port, handler)Start HTTP server

Basic Server

function handleRequest(req) {
    print("Received: " + req.method + " " + req.path);
    return "Hello from Unnarize!";
}

print("Server starting on port 8080...");
ucoreHttp.listen(8080, "handleRequest");

Request Object

The handler receives a request object with:

PropertyDescription
methodHTTP method (GET, POST, etc)
pathRequest path

Example with Routing

function router(req) {
    if (req.path == "/") {
        return "Welcome!";
    } else if (req.path == "/api") {
        return "{\"status\": \"ok\"}";
    }
    return "Not Found";
}

ucoreHttp.listen(8080, "router");
