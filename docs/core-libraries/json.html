ucoreJson

JSON parsing and serialization.

Functions

FunctionReturnsDescription
parse(jsonString) Map/Array Parse JSON string to value
stringify(value) string Serialize value to JSON string
readFile(path) Map/Array Read and parse JSON file
writeFile(path, value) bool Write value to JSON file

Basic Usage

parse / stringify

var jsonStr = '{"name":"Alice","age":30}';
var data = ucoreJson.parse(jsonStr);

print(data["name"]); // "Alice"
print(data["age"]);  // 30

// Modify and serialize
data["age"] = 31;
var output = ucoreJson.stringify(data);
print(output); // {"name":"Alice","age":31}

File Operations

// Read JSON file
var config = ucoreJson.readFile("config.json");
print(config["version"]);

// Write JSON file
var settings = map();
settings["theme"] = "dark";
settings["language"] = "en";
ucoreJson.writeFile("settings.json", settings);

Supported Types

  • Objects → Unnarize Maps
  • Arrays → Unnarize Arrays
  • Strings → Unnarize Strings
  • Numbers → Unnarize Numbers (int/float)
  • Booleans → Unnarize Booleans
  • null → Unnarize nil

Error Handling

Returns nil on parse errors.

var result = ucoreJson.parse("invalid json");
if (length(result) == 0) {
    print("Parse error!");
}
