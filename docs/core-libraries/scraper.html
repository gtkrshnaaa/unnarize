ucoreScraper

HTML parsing, CSS selectors, and web scraping.

Functions

FunctionReturnsDescription
fetch(url) string Fetch HTML from URL
download(url, filepath) bool Download HTML to file
select(html, selector) Array Select elements from HTML string
parseFile(path, selector) Array Parse file and select elements
parse(html, [debug]) nil Parse HTML (debug mode prints tree)

Supported CSS Selectors

  • tagname - Element selector (div, p, a)
  • .classname - Class selector
  • #id - ID selector
  • parent child - Descendant combinator

Basic Usage

Fetch and Select

var html = ucoreScraper.fetch("https://example.com");
var links = ucoreScraper.select(html, "a");

for (var i = 0; i < length(links); i = i + 1) {
    print(links[i]["text"]);
}

Download for Offline Processing

var url = "https://en.wikipedia.org/wiki/Main_Page";
ucoreScraper.download(url, "wikipedia.html");

// Later, process locally
var headlines = ucoreScraper.parseFile("wikipedia.html", ".mw-headline");
print("Found " + length(headlines) + " headlines");

Element Structure

Selected elements are returned as Maps with:

KeyTypeDescription
tagstringTag name (div, p, a)
textstringInner text content
attributesMapElement attributes
 
var links = ucoreScraper.select(html, "a");
var link = links[0];

print(link["tag"]);        // "a"
print(link["text"]);       // "Click here"
print(link["attributes"]["href"]); // "/page"
